cp2cg - cpuprofile to call graph

Convert a V8 .cpuprofile file to a package/module-scoped call graph. For more info, visit the GitHub repo: https://github.com/pmuellr/cp2cg

Drop a .cpuprofile file on this page to have it's call graph rendered, like the sample below. Here's a sample you can download: busy.cpuprofile
